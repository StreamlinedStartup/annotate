Home page

This simulates a single-page application navigating between routes via pushState. annotate.js should remain active and the toolbar should stay visible across route changes.

Each route change updates window.history and re-calls Annotate.refresh() to load per-page comments.

About page

This is the "about" route. Select text here or draw a rectangle to leave a comment. These comments should be kept separate from those on the Home page, since annotate.js namespaces by page key.

Pricing page

Pricing information would live here. This is a common page for UI feedback — reviewers can highlight copy, box misaligned elements, and pin questions to specific figures.

Contact page

Contact form and details. Any review annotations left here are stored under the /contact page key and do not mix with other pages.
